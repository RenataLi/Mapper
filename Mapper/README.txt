Доброго времени суток,немного о задании:
 - формат представления объектов:
   - обозначения самого объекта находится в {}
	 - массив находится в []
	 - перед типом обЪектов пишется его тип
         - значения типов находятся в ""
	 - обозначение простых типов перед их значениями
		 	"" or null or "String" - String
       			c - char
       			b - byte
       			s - short
       			i - int
       			l - long
       			f - float
       			d - double
       			B - boolean
       			t - LocalTime
       			D - LocalDate
       			T - LocalDateTime
       			0 - null (без кавычек)
	 - Пример на объекте Student(также представлен в одном из тестов)
	{org.test.Student
	"birthday" = "11 декабрь 1979 н.э.",
	"name" = "Petya",
	"rating" = d"99.5",
	"пол" = c"M"
	}
	В данном примере буквы c и d означают char и double соответственно.Это сделано для четкого 	определения типа.
 - реализованно сохранение и восстановление типов,record  а также присутствует реализация reatainIdentity
 - в директории test/java присутствуют тесты с различными примерами.Также для некторых тестах присутствуют тестовые классы,которые можно посмотреть в директории src/test Присутствуют примеры с потоками и файлами(файл file.txt - тестируемый файл) также.
 - вся реализация находится в другом не дефолтном пакете(не ru.hse.homework4).
 - также присутствуют обычные комментари в коде и javadoc.
